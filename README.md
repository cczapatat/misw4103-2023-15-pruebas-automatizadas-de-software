# Correr pruebas cypress
Modo headless
```
cd cypress
npx cypress run --browser chrome
```

Modo interactivo
```
cypress open
```

Seleccionar E2E, agregar el folder de cypress al arbol de proyectos,
Seleccionar el navegador chrome, seleccionar los escenarios a correr
